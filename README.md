# Train Times

This is an iOS app build in Expo and React native that will let users register the UK train services they're interested in, and then be able to see these at a glance.
It also has the ability to let the user add widgets to his iOS home screen, where the user can favourite certain services to see the next available trian times.

## Developer Notes

- **Target Platform**: iOS only (no Android support planned)
- **Development OS**: macOS

# Tech Stack

1. Git
1. React Native
1. Zustand
1. Nativewind for styling
1. Tanstack query
1. Tanstack table
1. Swift
1. iOS
1. Login with Google
1. Login with Apple
1. Backend server in nodejs (for the user to post favourites)
1. Database in mysql (to store user favourites)

The application uses the [Bulletproof react](https://github.com/alan2207/bulletproof-react) architecture of the front end
